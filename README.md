# Backend Go — Изучение Go по книге Цукалоса

Репозиторий для изучения Go по книге **"Golang для профи: работа с сетью, многопоточность, структуры данных и машинное обучение с Go"** (Михалис Цукалос).

## Цель

За 6-8 недель освоить Go и выйти на уровень Junior/Middle Backend разработчика.

## Структура проекта

```
BackendGo/
├── go.mod              # Go модуль
├── README.md           # Этот файл
├── Motivations.md      # Система мотивации и геймификация
├── коуча.md            # План работы с коучем
│
├── book/               # Код по книге Цукалоса
│   ├── ch03/           # Глава 3: Основные типы данных
│   │   ├── examples/   # Примеры из книги
│   │   └── exercises/  # Мои решения упражнений
│   ├── ch04/           # Глава 4: Массивы, слайсы, мапы
│   ├── ch05/           # Глава 5: Структуры
│   ├── ch06/           # Глава 6: Goroutines
│   ├── ch07/           # Глава 7: Channels
│   └── ...
│
└── projects/           # Практические проекты
    └── task-api/       # Task Management API (будущий)
```

## Как запускать

### Примеры из книги

```bash
go run ./book/ch03/examples/types.go
```

### Мои упражнения

```bash
go run ./book/ch03/exercises/ex01.go
```

### Тесты (когда появятся)

```bash
go test ./book/ch03/...
```
